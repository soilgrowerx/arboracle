# Contribute to the Arboracle Knowledge Base

We welcome contributions from arborists, foresters, researchers, and enthusiasts to help grow the Arboracle Knowledge Base. Your expertise can help cultivate knowledge and preserve nature for tomorrow's forest legacy.

## How to Contribute

1.  **Fork the Repository:** Our knowledge base content is managed in Markdown files within our GitHub repository. Start by forking the `soilgrowerx/arboracle` repository.

2.  **Create New Content or Edit Existing:**
    *   **New Articles:** For new topics, create a new Markdown file (`.md`) in the `src/app/knowledge` directory. Use a clear, descriptive filename (e.g., `tree-diseases-overview.md`).
    *   **Edits:** To improve existing articles, navigate to the relevant Markdown file and make your changes.

3.  **Follow Markdown Standards:**
    *   Use `#` for main headings, `##` for subheadings, and so on.
    *   Use `*` or `-` for bullet points.
    *   Use `1.` for numbered lists.
    *   For code blocks, use triple backticks (``````) with the language specified (e.g., ````javascript`).

4.  **Add Citations (if applicable):** If you reference research papers, studies, or external resources, please include citations or links.

5.  **Submit a Pull Request:** Once your changes are ready, commit them to your forked repository and open a Pull Request (PR) to the `main` branch of the `soilgrowerx/arboracle` repository. Provide a clear description of your changes in the PR.

## Content Guidelines

*   **Accuracy:** All information must be scientifically accurate and up-to-date.
*   **Clarity:** Write in clear, concise language that is accessible to a broad audience, from students to seasoned professionals.
*   **Relevance:** Content should directly relate to arboriculture, forestry, tree health, ecological insights, or sustainable land management.
*   **Originality:** While you can reference external sources, ensure your contributions are original or properly attributed.

## Review Process

All Pull Requests will be reviewed by our team of experts. We may suggest revisions or ask for clarifications to ensure the quality and accuracy of the content.

Thank you for your contribution to Arboracle!
